This is only for road all ambulance detection
To train the model
1) Connect to google colab
2) Upload datasets to drive
3) Open the file inside this folder (.ipnyb)
4) Change path to dataset 
5) Upload custom_data.yaml from this folder to colab (yolov5/data)
6) run the model

7) Make sure to run command using colab and change dir to /yolov5
